# betterthanclay-dark-syntax theme

Dark syntax theme for Atom with focus on PHP and CSS.

### With support for:

* PHP
* HTML
* CSS
* Less
* Sass
* Javascript
* JSON
* Yaml
* Neon
* Markdown
